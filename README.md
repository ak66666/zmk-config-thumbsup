# zmk-config
This ZMK setup drives my Thumbs Up! v5 (also applicable to v8, v8b, v11) and v9 keyboards.
  
https://github.com/ak66666

Configuration supports Cirque ProGlide Touchpad and has ZMK.Studion support enabled.
